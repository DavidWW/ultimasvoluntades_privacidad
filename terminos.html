1. Descripción del Servicio

Últimas Voluntades es una aplicación móvil que permite a los usuarios:

  • Registrar sus deseos y preferencias personales para después de su fallecimiento
  • Almacenar mensajes, fotos, audios y documentos para destinatarios específicos
  • Configurar un sistema de acceso diferido para dicho contenido

3. Requisitos de Uso

Para utilizar Últimas Voluntades debes:

  • Ser mayor de 18 años
  • Disponer de una cuenta de Apple ID válida
